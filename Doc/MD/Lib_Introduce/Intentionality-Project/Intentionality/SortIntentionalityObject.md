# SortIntentionalityObject

***

映射SortIntentionalityObject接受空元素集合（NullElementSet）为定义域，意向性对象集合（IntentionalityObjectSet），定义为：

    Map SortIntentionalityObject(&NullElementSet, &IntentionalityObjectSet, SortIntentionalityObjectFunction);

意向性对象排序映射（SortIntentionalityObject）在值域意向性对象元素上执行排序映射。该排序将会用于优化检索。

关于意向性对象的测试案例，跳转至[Intentionality案例](Intentionality_Case.md)
