# RemoveIntentionalityObject

***

映射RemoveIntentionalityObject接受超索引元素集合（SuperIndexSet）为定义域，意向性对象集合（IntentionalityObjectSet）为值域，定义为：

    Map RemoveIntentionalityObject(&SuperIndexSet, &IntentionalityObjectSet, RemoveIntentionalityObjectFunction);

意向性对象移除映射（RemoveIntentionalityObject）在值域意向性对象上移除定义域超索引代表的意向性。

关于意向性对象的测试案例，跳转至[Intentionality案例](Intentionality_Case.md)
