# ElementEnvironmentSearcher

***

集合ElementEnvironmentSearcher为所有总元素环境检索器的集合，其定义如下：

    Set ElementEnvironmentSearcher(EmptyElement, ElementEnvironmentSearcherTest);

总元素环境检索器集合（ElemenetEnvironmentSearcher）由所有总元素环境检索器元素组成，用于总元素环境检索器相关操作。

总元素环境检索器元素作为个体智能体对自身系统所有实体元素索引的映射关系的管理实体，相关操作需要由其集合进行指明。

关于总元素环境检索器测试案例，跳转至[EES_Base案例](EES_Base_Case.md)。
