# LexOrderEES

***

集合LexOrderEES为所有字典序子元素检索器的集合，定义如下：

    Set LexOrderEES(EmptyElement, LexOrderEESTest);

字典序子元素环境检索器集合（LexOrderEES）由所有字典序子元素检索器元素组成，用于字典序子元素检索器相关操作。其操作可通过宏GetEleOrder和命名空间SubEES_LexOrder完成。

关于如何构建并使用字典序子元素环境检索器，跳转至[SubEES_LexOrder_Case](SubEES_LexOrder_Case.md)，或者[EES_Base案例](..\EES_Base\EES_Base_Case.md)。
