# SubEES_LexOrder

***

字典序子元素环境检索器是子元素环境检索器的一种，其负责管理流检索信息到中间层超索引信息的映射关系。字典序子元素环境检索器不允许重复的映射关系，按照字典序进行排序，属于变长容器。
