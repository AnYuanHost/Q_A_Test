# ElementEnvironmentGetSuperIndex

***

映射ElementEnvironmentGetSuperIndex接受元素环境集合（ElementEnvironmentSet）为定义域，空元素集合（NullElementSet）为值域，定义如下：

    Map ElementEnvironmentGetSuperIndex(&ElementEnvironmentSet, &NullElementSet, ElementEnvironmentGetSuperIndexFunction);

元素环境获取超索引映射（ElementEnvironmentGetSuperIndex）从定义域元素环境元素中获取超索引元素，将其转交传递到值域空元素上。

关于元素环境如何完成超索引获取，跳转至[EES_Base案例](..\EES_Base\EES_Base_Case.md)。
