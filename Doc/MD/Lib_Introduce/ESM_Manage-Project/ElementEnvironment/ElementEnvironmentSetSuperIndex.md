# ElementEnvironmentSetSuperIndex

***

映射ElementEnvironmentSetSuperIndex接受超索引集合（SuperIndexSet）为定义域，元素环境集合（ElementEnvironmentSet）为值域，定义如下：

    Map ElementEnvironmentSetSuperIndex(&SuperIndexSet, &ElementEnvironmentSet, ElementEnvironemntSetSuperIndexFunction);

元素环境设置超索引映射（ElementEnvironmentSetSuperIndex）将定义域超索引元素转交传递到值域元素环境的对应位置上。

关于元素环境如何完成超索引信息设置，跳转至[EES_Base案例](..\EES_Base\EES_Base_Case.md)。
