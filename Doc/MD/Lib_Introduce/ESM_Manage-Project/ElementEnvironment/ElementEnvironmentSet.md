# ElementEnvironmentSet

***

集合ElementEnvironmentSet为所有元素环境的集合，定义如下：

    Set ElementEnvironmentSet(EmptyElement, ElementEnvironmentSetFunction);

元素环境集合（ElementEnvironmentSet）由所有元素环境元素组成，用于元素环境的相关映射。

元素环境需要完成对智能体中几乎所有元素实体的管理工作，其需要提供足够完善的容器管理机制。

关于元素环境测试案例，跳转至[EES_Base案例](..\EES_Base\EES_Base_Case.md)。
