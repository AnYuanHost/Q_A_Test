# ElementEnvironmentExpand

***

映射ElementEnvironmentExpand接受空元素集合（NullElementSet）为定义域，元素环境集合（ElementEnvironmentSet）为值域，定义如下：

    Map ElementEnvironmentExpand(&NullElementSet, &ElementEnvironemntSet, ElementEnvironemntExpandFunction);

元素环境扩容映射（ElementEnvironmentExpand）在值域元素环境元素上完成扩容操作。

关于元素环境如何完成扩容，跳转至[EES_Base案例](..\EES_Base\EES_Base_Case.md)。
