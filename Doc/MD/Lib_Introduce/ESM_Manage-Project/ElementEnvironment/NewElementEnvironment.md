# NewElementEnvironment

***

映射NewElementEnvironment接受空元素集合（NullElementSet）为定义域和值域，定义如下：

    Map NewElementEnvironment(&NullElementSet, &NullElementSet, NewElementEnvironmentFunction);

新建元素环境映射（NewElementEnvironment）在值域空元素上构造元素环境元素。该构造过程包括默认的参数构建。

关于元素环境如何完成新建，跳转至[EES_Base案例](..\EES_Base\EES_Base_Case.md)。
