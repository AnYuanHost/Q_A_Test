# ReleaseSuperIndexSubEES

***

映射ReleaseSuperIndexSubEES接受空元素集合（NullElementSet）为定义域，超索引子元素环境检索器集合（SuperIndexSubEES）为值域，定义为：

    Map ReleaseSuperIndexSubEES(&NullElementSet, &SuperIndexSubEES, RelaseSuperIndexSubEESFunction);

超索引子元素环境检索器释放映射（ReleaseSuperIndexSubEES）分析并释放值域超索引子元素环境检索器元素。

关于如何完成超索引子元素环境检索器释放，跳转至[SubEES_SuperIndex案例](SubEES_SuperIndex_Case.md)，或者[EES_Base案例](..\EES_Base\EES_Base_Case.md)。
