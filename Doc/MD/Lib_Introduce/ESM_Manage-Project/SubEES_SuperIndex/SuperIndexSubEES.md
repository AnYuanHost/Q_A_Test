# SuperIndexSubEES

***

集合SuperIndexSubEES为所有超索引子元素环境检索器的集合，其定义为：

    Set SUperIndexSubEES(EmptyElement, LexOrderEESTest);

超索引子元素环境检索器集合（SuperIndexSubEES）由所有超索引子元素环境检索器元素组成，用于超索引子元素环境检索器相关操作。其操作可通过宏GetEleOrder和命名空间SuperIndexSubEES_Struct完成。

关于如何构建并使用超索引子元素环境检索器，跳转至[SubEES_SuperIndex案例](SubEES_SuperIndex_Case.md)，或者[EES_Base案例](..\EES_Base\EES_Base_Case.md)。
