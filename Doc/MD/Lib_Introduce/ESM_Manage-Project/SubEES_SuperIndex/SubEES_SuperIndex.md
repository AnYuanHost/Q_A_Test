# SubEES_SuperIndex

***

超索引子元素环境检索器是子元素环境检索器的一种，其负责管理超索引信息到中间层超索引信息的映射关系。超索引子元素环境检索器不允许重复的映射关系，其按照超索引大小顺序排序，属于变长容器。
