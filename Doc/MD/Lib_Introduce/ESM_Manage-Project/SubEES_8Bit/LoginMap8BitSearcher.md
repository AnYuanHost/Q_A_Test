# LoginMap8BitSearcher

***

映射LoginMap8BitSearcher接受空元素集合（NullElementSet）为定义域和值域，定义如下：

    Map LoginMap8BitSearcher(&NullElementSet, &NullElementSet, LoginMap8BitSearcherFunction);

注册8比特子元素环境检索器映射映射（LoginMap8BitSearcher）在值域空元素上按照总元素环境检索器给出的映射集要求，构造8比特子元素环境检索器映射集。

关于如何构造8比特子元素环境检索器映射集，跳转至[SubEES_8Bit](SubEES_8Bit_Case.md)，或者[EES_Base案例](..\EES_Base\EES_Base_Case.md)。
