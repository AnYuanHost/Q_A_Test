# New8BitElement

***

映射New8BitElement接受空元素集合（NullElementSet）为定义域和值域，其定义如下：

    Map New8BitElement(&NullElementSet, &NullElementSet, &New8BitElementFunction);

新建8比特元素映射（New8BitElement）在值域空元素上创建8比特元素，并初始化为0。

关于如何使用新建8比特元素映射完成构建并使用，跳转至[SubEES_8Bite](SubEES_8Bit_Case.md)，或者[EES_Base案例](..\EES_Base\EES_Base_Case.md)。
