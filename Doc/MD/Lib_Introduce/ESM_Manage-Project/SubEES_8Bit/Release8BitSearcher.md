# Release8BitSearcher

***

映射Release8BitSearcher接受空元素集合（NullElementSet）为定义域，8比特子元素环境检索器集合（_8BitSearcher）为值域，定义如下：

    Map Release8BitSearcher(&NullElementSet, &_8BitSearcher, Release8BitSearcherFunction);

释放8比特子元素环境检索器映射（Release8BitSearcher）释放值域8比特子元素环境检索器元素。

关于如何释放8比特子元素环境检索器，跳转至[SubEES_8Bit](SubEES_8Bit_Case.md)，或者[EES_Base案例](..\EES_Base\EES_Base_Case.md)。
