# SubEES_8Bit

***

8比特子元素环境检索器是子元素环境检索器的一种，其负责管理8比特检索信息到中间层超索引信息的映射关系。8比特子元素环境检索器是定长容器，每个槽位只接受一个固定8比特信息，不执行8比特的排序操作。
