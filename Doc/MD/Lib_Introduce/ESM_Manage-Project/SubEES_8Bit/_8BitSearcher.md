# _8BitSearcher

***

集合_8BitSearcher为所有8比特子元素检索器的集合，定义如下：

    Set _8BitSearcher(EmptyElement, _8BitSearcherTest);

8比特子元素环境检索器集合（_8BitSearcher）由所有8比特子元素检索器元素组成，用于8比特子元素检索器相关操作。其元素可通过宏GetEleOrder和命名空间SubEES_8Bit完成。

关于如何构建并使用8比特子元素环境检索器，跳转至[SubEES_8Bit](SubEES_8Bit_Case.md)，或者[EES_Base案例](..\EES_Base\EES_Base_Case.md)。
