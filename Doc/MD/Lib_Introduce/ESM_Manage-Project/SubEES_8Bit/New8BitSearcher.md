# New8BitSearcher

***

映射New8BitSearcher接受空元素集合（NullElementSet）为定义域和值域，定义如下：

    Map New8BitSearcher(&NullElementSet, &NullElementSet, New8BitSearcherFunction);

新建8比特子元素环境检索器映射（New8BitSearcher）在值域空元素上构造并初始化8比特子元素环境检索器元素。

关于如何构建8比特子元素环境检索器，跳转至[SubEES_8Bit](SubEES_8Bit_Case.md)，或者[EES_Base案例](..\EES_Base\EES_Base_Case.md)。
