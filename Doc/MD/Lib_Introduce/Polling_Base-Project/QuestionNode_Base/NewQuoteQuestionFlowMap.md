# NewQuoteQuestionFlowMap

***

映射NewQuoteQuestionFlowMap接受空元素集合（NullElementSet）为定义域和值域，定义为：

    Map NewQuoteQuestionFlowMap(&NullElementSet, &NullElementSet, NewQuoteQuestionFlowMapFunction);

新建引用问答流图映射（NewQuoteQuestionFlowMap）在值域空元素上构造一个默认的引用问答流图元素。

关于如何构造引用问答流图元素，参考案例[QuestionNode_Base案例](QuestionNode_Base_Case.md)
