# SetQuestionNodeDomain

***

映射SetQuestionNodeDomain接受问答节点参数节点集合（QuestionParameterNode）为定义域，问答节点集合（QuestionNodeSet）为值域，定义为：

    Map SetQuestionNodeDomain(&QuestionParameterNode, &QuestionNodeSet, SetQuestionNodeDomainFunction);

问答节点设置定义域映射（SetQuestionNodeDomain）将定义域问答节点参数节点元素转交传递到值域问答节点的对应位置。

关于如何完成问答节点定义域信息设置，参考案例[QuestionNode_Base案例](QuestionNode_Base_Case.md)
