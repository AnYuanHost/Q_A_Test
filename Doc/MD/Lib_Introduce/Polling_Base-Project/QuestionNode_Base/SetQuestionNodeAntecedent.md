# SetQuestionNodeAntecedent

***

映射SetQuestionNodeAntecedent接受元素序列集合（ElementOrderSet）为定义域，问答节点集合（QuestionNodeSet）为值域，定义为：

    Map SetQuestionNodeAntecedent(&ElementOrderSet, &QuestionNodeSet, SetQuestionNodeAntecedentFunction);

问答节点设置前置问答序列映射（SetQuestionNodeAntecedent）将定义域的问答节点信息序列设置到值域问答节点元素的对应位置上。

关于如何完成问答节点参数节点的前置节点设置，参考案例[QuestionNode_Base案例](QuestionNode_Base_Case.md)
