# DefragmentQuestionFlowMap

***

映射DefragmentQuestionFlowMap接受空元素集合（NullElementSet）为定义域，问答流图集合（QuestionFlowMap）为值域，定义为：

    Map DefragmentQuestionFlowMap(&NullElementSet, &QuestionFlowMap, DefragmentQuestionFlowMapFunction);

问答流图碎片整理映射（DefragmentQuestionFlowMap）在值域问答流图元素上完成碎片整理操作。

关于如何完成问答流图碎片整理，参考案例[QuestionNode_Base案例](QuestionNode_Base_Case.md)
