# QuestionFlowMapReplacePackage

***

集合QuestionFlowMapReplacePackage为所有问答流图替换包的集合，定义为：

    Set QuestionFlowMapReplacePackage(EmptyElement, QuestionFlowMapReplacePackageTest);

问答流图替换包集合（QuestionFlowMapReplacePackage）由所有问答流图替换包元素组成，用于问答流图替换包元素的操作。

关于如何使用问答流图替换包集合，参考案例[QuestionNode_Base案例](QuestionNode_Base_Case.md)
