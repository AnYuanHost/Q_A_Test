# SetQuestionNodeSuperIndex

***

映射SetQuestionNodeSuperIndex接受超索引集合（SuperIndexSet）为定义域，问答流图节点集合（QuestionNodeSet）为值域，定义为：

    Map SetQuestionNodeSuperIndex(&SuperIndexSet, &QuestionNodeSet, SetQuestionNodeSuperIndexFunction);

问答节点设置超索引映射（SetQuestionNodeSuperIndex）将定义域超索引信息设置到值域问答流图节点元素的指定位置。

关于如何完成问答节点超索引设置，参考案例[QuestionNode_Base案例](QuestionNode_Base_Case.md)
