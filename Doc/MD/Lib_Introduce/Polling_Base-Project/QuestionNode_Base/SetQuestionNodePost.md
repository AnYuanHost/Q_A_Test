# SetQuestionNodePost

***

映射SetQuestionNodePost接受元素序列集合（ElementOrderSet）为定义域，问答节点集合（QuestionNodeSet）为值域，定义为：

    Map SetQuestionNodePost(&ElementOrderSet, &QuestionNodeSet, SetQuestionNodePostFunction);

问答节点设置后置问答节点序列映射（SetQuestionNodePost）将定义域的问答节点信息序列设置到值域问答节点元素的对应位置上。

关于如何完成问答节点后置问答节点序列设置，参考案例[QuestionNode_Base案例](QuestionNode_Base_Case.md)
