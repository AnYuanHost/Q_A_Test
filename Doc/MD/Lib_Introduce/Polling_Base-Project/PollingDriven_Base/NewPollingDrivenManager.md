# NewPollingDrivenManager

***

映射NewPollingDrivenManager接受空元素集合（NullElementSet）为定义域和值域，定义为：

    Map NewPollingDrivenManager(&NullElementSet, &NullElementSet, NewPollingDrivenManagerFunction);

新建轮询机管理映射（NewPollingDrivenManager）在值域空元素上使用默认方法构造一个轮询机管理元素。

关于如何完成轮询机管理构造，跳转至[PollingDriven_Base案例](PollingDriven_Base_Case.md)
