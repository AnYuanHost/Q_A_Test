# SetEESPollingDriven

***

映射SetEESPollingDriven接受元素环境检索器集合（ElementEnvironmentSearcher）为定义域，轮询机集合（PollingDrivenSet）为值域，定义为：

    Map SetEESPollingDriven(&ElementEnvironmentSearcher, &PollingDrivenSet, SetEESPollingDrivenFunction);

轮询机设置元素环境检索器映射（SetEESPollingDriven）将定义域元素环境检索器设置到值域轮询机的指定位置上。

关于如何完成轮询机元素环境检索器设置，跳转至[PollingDriven_Base案例](PollingDriven_Base_Case.md)
