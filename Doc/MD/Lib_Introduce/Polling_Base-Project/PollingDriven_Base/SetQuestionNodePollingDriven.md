# SetQuestionNodePollingDriven

***

映射SetQuestionNodePollingDriven接受问答节点集合（QuestionNodeSet）为定义域，轮询机集合（PollingDrivenSet）为值域，定义为：

    Map SetQuestionNodePollingDriven(&QuestionNodeSet, &PollingDrivenSet, SetQuestionNodePollingDrivenFunction);

轮询机问答节点设置映射（SetQuestionNodePollingDriven）将定义域问答节点元素设置到值域轮询机元素。

关于如何完成轮询机轮询机问答节点设置，跳转至[PollingDriven_Base案例](PollingDriven_Base_Case.md)
