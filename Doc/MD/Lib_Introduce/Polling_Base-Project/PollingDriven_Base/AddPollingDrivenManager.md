# AddPollingDrivenManager

***

映射AddPollingDrivenManager接受轮询机集合（PollingDrivenSet）为定义域，轮询机管理集合（PollingDrivenManager）为值域，定义为：

    Map AddPollingDrivenManager(&PollingDrivenSet, &PollingDrivenManager, PollingDrivenManagerFunction);

轮询机管理添加映射（AddPollingDrivenManager）将定义域轮询机元素加入到值域轮询机管理元素中，

关于如何完成轮询机管理添加轮询机，跳转至[PollingDriven_Base案例](PollingDriven_Base_Case.md)
