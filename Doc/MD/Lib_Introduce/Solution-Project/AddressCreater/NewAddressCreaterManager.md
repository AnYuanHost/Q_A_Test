# NewAddressCreaterManager

***

映射NewAddressCreaterManager接受空元素集合（NullElementSet）为定义域和值域，定义为：

    Map NewAddressCreaterManager(&NullElementSet, &NullElementSet, NewAddressCreaterManagerFunction);

新建地址生成器管理映射（NewAddressCreaterManager）在值域空元素上构造默认的地址生成器管理。

关于如何完成地址生成管理器构造，参考[AddressCreater案例](AddressCreater_Case.md)
