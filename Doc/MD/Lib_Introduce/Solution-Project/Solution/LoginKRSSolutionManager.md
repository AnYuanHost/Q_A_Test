# LoginKRSSolutionManager

***

映射LoginKRSSolutionManager接受解决方案集合（SolutionSet）为定义域，知识库解决方案管理集合（KRSSolutionManager），定义为：

    Map LoginKRSSolutionManager(&SolutionSet, &KRSSolutionManager, LoginKRSSolutionManagerFunction);

知识库解决方案管理注册映射（LoginKRSSolutionManager）将定义域解决方案元素注册到值域知识库解决方案管理元素中。

关于如何完成知识库解决方案管理注册，参考[Solution案例](Solution_Case.md)
