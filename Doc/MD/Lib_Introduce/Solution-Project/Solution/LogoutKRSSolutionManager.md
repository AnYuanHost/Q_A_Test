# LogoutKRSSolutionManager

***

映射LogoutKRSSolutionManager接受无符号整型集合（UIntSet）为定义域，知识库解决方案管理集合（KRSSolutionManager）为值域，定义为：

    Map LogoutKRSSolutionManager(&UIntSet, &KRSSolutionManager, LogoutKRSSolutionManagerFunction);

知识库解决方案管理注销映射（LogoutKRSSolutionManager）在值域知识库解决方案管理元素中在无符号整型元素地址信息上完成检索和注销。

关于如何完成知识库解决方案管理注销，参考[Solution案例](Solution_Case.md)
