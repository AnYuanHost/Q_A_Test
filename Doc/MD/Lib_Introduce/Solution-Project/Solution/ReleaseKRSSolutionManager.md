# ReleaseKRSSolutionManager

***

映射ReleaseKRSSolutionManager接受空元素集合（NullElementSet）为定义域，知识库解决方案管理集合（KRSSolutionManager）为值域，定义为：

    Map ReleaseKRSSolutionManager(&NullElementSet, &KRSSolutionManager, ReleaseKRSSolutionManagerFuncion);

知识库解决方案管理释放映射（ReleaseKRSSolutionManager）分析并释放值域知识库解决方案管理元素。

关于如何完成知识库解决方案管理释放，参考[Solution案例](Solution_Case.md)
