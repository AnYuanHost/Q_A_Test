# DefragemntKRSSolutionManager

***

映射DefragmentKRSSolutionManager接受空元素集合（NullElementSet）为定义域，知识库解决方案管理集合（KRSSolutionManager）为值域，定义为：

    Map DefragmentKRSSolutionManager(&NullElementSet, &KRSSolutionManager, DefragmentKRSSolutionManagerFunction);

知识库解决方案管理碎片整理映射（DefragmentKRSSolutionManager）在值域知识库解决方案管理元素上进行碎片整理。

关于如何完成知识库解决方案管理碎片整理，参考[Solution案例](Solution_Case.md)
