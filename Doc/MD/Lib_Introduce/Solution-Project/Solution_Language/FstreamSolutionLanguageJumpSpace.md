# FstreamSolutionLanguageJumpSpace

***

映射FstreamSolutionLanguageJumpSpace接受空元素集合（NullElementSet）为定义域，文件集合（FstreamSet）为值域，定义为：

    Map FstreamSolutionLanguageJumpSpace(&NullElementSet, &FstreamSet, FstreamSoluitionLanguageJumpSpaceFunction);

解决方案语言文件跳过空白符映射（FstreamSolutionLanguageJumpSpace）在值域文件元素上跳过被标记的空白符。

关于如何完成解决方案文件语言跳过空白符，参考[Solution_Language案例](Solution_Language_Case.md)
