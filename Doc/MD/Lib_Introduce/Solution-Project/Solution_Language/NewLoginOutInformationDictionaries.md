# NewLoginOutInformationDictionaries

***

映射NewLoginOutInformationDictionaries接受空元素集合（NullElementSet）为定义域和值域，定义为：

    Map NewLoginOutInformationDictionaries(&NullElementSet, &NullElementSet, NewLoginOutInformationDictionariesFunction);

新建注册销信息字典映射（NewLoginOutInformationDictionaries）在值域空元素上默认构造注册销信息字典元素。

关于如何完成注册销信息字典构造，参考[Solution_Language案例](Solution_Language_Case.md)
