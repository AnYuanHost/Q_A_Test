# ReleaseSolutionLanguageNodeOrder

***

映射ReleaseSolutionLanguageNodeOrder接受空元素集合（NullElementSet）为定义域，解决方案节点序列集合（SolutionLanguageNodeOrder）为值域，定义为：

    Map ReleaseSolutionLanguageNodeOrder(&NullElementSet, &SolutionLanguageNodeOrder, ReleaseSolutionLanguageNodeOrderFunction);

释放解决方案语言节点序列映射（ReleaseSolutionLanguageNodeOrder）分析并释放值域解决方案节点序列元素。

关于如何完成解决方案文件语言节点序列释放，参考[Solution_Language案例](Solution_Language_Case.md)
