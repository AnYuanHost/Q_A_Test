# SolutionLanguageExplainManagerSet

***

集合SolutionLanguageExplainManagerSet为解决方案语言描述管理的集合，定义为：

    Set SolutionLanguageExplainManagerSet(EmptyElement, SolutionLanguageExplainManagerSetTest);

解决方案描述管理集合（SolutionLanguageExplainManagerSet）由所有解决方案语言描述管理元素组成，用于解决方案语言描述管理相关操作。

关于如何使用解决方案文件语言描述管理集合，参考[Solution_Language案例](Solution_Language_Case.md)
