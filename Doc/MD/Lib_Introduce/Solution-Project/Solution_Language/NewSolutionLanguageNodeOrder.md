# NewSolutionLanguageNodeOrder

***

映射NewSolutionLanguageNodeOrder接受空元素集合（NullElementSet）为定义域和值域，定义为：

    Map NewSolutionLanguageNodeOrder(&NullElementSet, &NullElementSet, NewSolutionLanguageNodeOrderFunction);

新建解决方案语言节点序列映射（NewSolutionLanguageNodeOrder）在值域空元素上默认构造解决方案语言节点序列元素。

关于如何构造解决方案文件语言节点序列，参考[Solution_Language案例](Solution_Language_Case.md)
