# FstreamSolutionLanguageMapEndTest

***

映射FstreamSolutionLanguageMapEndTest接受文件集合（FstreamSet）为定义域，布尔类型集合（BoolSet）为值域，定义为：

    Map FstreamSolutionLanguageMapEndTest(&FstreamSet, &BoolSet, FstreamSolutionLanguageMapEndTestFunction);

解决方案语言文件图末尾检测映射（FstreamSolutionLanguageMapEndTest）在定义域文件元素上进行图末尾检测，检测是否为图结构的末尾，并将检测结果赋值给值域布尔元素。

关于如何完成解决方案文件语言图末尾检测，参考[Solution_Language案例](Solution_Language_Case.md)
