# NewSolutionLanguageExplainManager

***

映射NewSolutionLanguageExplainManager接受空元素集合（NullElementSet）为定义域和值域，定义为：

    Map NewSolutionLanguageExplainManager(&NullElementSet, &NullElementSet, NewSolutionLanguageExplainManagerFunction);

新建解决方案语言描述管理映射（NewSolutionLanguageExplainManager）在值域空元素上默认构造解决方案语言描述管理元素。

关于如何完成解决方案文件语言描述管理构造，参考[Solution_Language案例](Solution_Language_Case.md)
