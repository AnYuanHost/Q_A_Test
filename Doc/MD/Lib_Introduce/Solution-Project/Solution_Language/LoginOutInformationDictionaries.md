# LoginOutInformationDictionaries

***

集合LoginOutInformationDictionaries为所有注册销信息字典的集合，定义为：

    Set LoginOutInformationDictionaries(EmptyElement, LoginOutInformationDictionariesTest);

注册销信息字典集合（LoginOutInformationDictionaries）由所有注册销信息字典元素组成，用于注册销信息字典的相关操作。

关于如何使用注册销信息字典集合，参考[Solution_Language案例](Solution_Language_Case.md)
