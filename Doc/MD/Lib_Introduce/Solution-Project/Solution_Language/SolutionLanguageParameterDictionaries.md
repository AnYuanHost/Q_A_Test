# SolutionLanguageParameterDictionaries

***

集合SolutionLanguageParameterDictionaries为解决方案语言参数字典的集合，定义为：

    Map SolutionLanguageParameterDictionaries(EmptyElement, SolutionLanguageParameterDictionariesTest);

解决方案语言参数字典集合（SolutionLanguageParameterDictionaries）由所有解决方案语言参数字典元素组成，用于解决方案语言参数字典相关操作。

关于如何使用解决方案文件语言参数字典，参考[Solution_Language案例](Solution_Language_Case.md)
