# LoginOutInformationDictionariesToData

***

映射LoginOutInformationDictionariesToData接受注册销字典集合（LoginOutInfortmaionDictionaries）为定义域，空元素集合（NullElementSet）为值域，定义为：

    Map LoginOutInformationDictionariesToData(&LoginOutInformationDictionaries, &NullElementSet, LoginOutInformationDictionariesToDataFunction);

注册销字典数据转化映射（LoginOutInformationDictionariesToData）分析定义域注册销字典元素信息，将内部信息转化为流信息，置于值域流元素上。

关于如何完成注册销字典的数据转化，参考[Solution_Language案例](Solution_Language_Case.md)
