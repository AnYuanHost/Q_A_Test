# ReleaseLoginOutInformationDictionaries

***

映射ReleaseLoginOutInformationDictionaries接受空元素集合（NullElementSet）为定义域，注册销信息字典集合（LoginOutInformationDictionaries）为值域，定义为：

    Map ReleaseLoginOutInfomrationDictionaries(&NullElementSet, &LoginOUtInformationDictionaries, ReleaseLoginOutInformatyionDictionariesFunction);

释放注册销信息字典映射（ReleaseLoginOutInformationDictionaires）分析并释放值域注册销信息字典元素。

关于如何完成注册销字典释放，参考[Solution_Language案例](Solution_Language_Case.md)
