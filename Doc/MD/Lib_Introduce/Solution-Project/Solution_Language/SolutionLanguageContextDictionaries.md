# SolutionLanguageContextDictionaries

***

集合SolutionLanguageContextDictionaries为解决方案语言语境字典的集合，定义为：

    Set SolutionLanguageContextDictionaries(EmptyElement, SolutionLanguageContextDictionariesTest);

解决方案语言语境字典集合（SolutionLanguageContextDictionaries）由所有解决方案语言语境字典元素组成，用于解决方案语言语境字典相关操作。

关于如何使用解决方案语言语境字典集合，参考[Solution_Language案例](Solution_Language_Case.md)
