# FstreamConstructSolutionInfomation

***

映射FstreamConstructSolutionInformation接受文件集合（FstreamSet）为定义域，解决方案语言表达管理集合（SolutionLanguageExplainManagerSet）为值域，定义为：

    Map FstreamConstructSolutionInformation(&FstreamSet, &SolutionLanguageExplainManagerSet, FstreamConstructSolutionInformation);

文件构造解决方案信息映射（FstreamConstructSolutionInformation）在定义域文件元素流中获得信息，在值域解决方案语言表达管理元素上构造参数相关的信息。

关于如何完成解决方案语言描述管理信息构造，参考[Solution_Language案例](Solution_Language_Case.md)
