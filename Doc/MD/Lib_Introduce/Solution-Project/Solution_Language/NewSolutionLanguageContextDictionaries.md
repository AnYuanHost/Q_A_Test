# NewSolutionLanguageContextDictionaries

***

映射NewSolutionLanguageContextDictionaries接受空元素集合（NullElementSet）为定义域和值域，定义为：

    Map NewSolutionLanguageContextDictionaries(&NullElementSet, &NullElementSet, NewSolutionLanguageContextDictionariesFunction);

新建解决方案语言语境字典映射（NewSolutionLanguageContextDictionaries）在值域空元素上默认构造解决方案语言语境字典元素。

关于如何完成解决方案文件语言语境字典构造，参考[Solution_Language案例](Solution_Language_Case.md)
