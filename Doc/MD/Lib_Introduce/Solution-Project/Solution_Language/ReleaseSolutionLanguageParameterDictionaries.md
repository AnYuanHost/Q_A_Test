# ReleaseSolutionLanguageParameterDictionaries

***

映射ReleaseSolutionLanguageParameterDictionaries接受空元素集合（NullElementSet）为定义域，解决方案语言参数字典集合（SolutionLanguageParameterDcitionaries）为值域，定义为：

    Map RleaseSolutionLanguageParameterDictionaries(&NullElementSet, &SolutionLanguageParameterDictionaries, ReleaseSolutionLanguageParameterDictionariesFunction);

释放解决方案语言参数字典映射（ReleaseSolutionLanguageParameterDictionaries）分析并释放值域解决方案语言参数字典元素。

关于如何完成解决方案文件语言参数字典释放，参考[Solution_Language案例](Solution_Language_Case.md)
