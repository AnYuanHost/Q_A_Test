# SolutionLanguageNodeOrder

***

集合SolutionLanguageNodeOrder为所有解决方案语言节点序列的集合，定义为：

    Set SolutionLanguageNodeOrder(EmptyElement, SolutionLanguageNodeOrderTest);

解决方案语言节点序列集合（SolutionLanguageNodeOrder）由所有解决方案语言节点序列元素组成，用于解决方案语言节点序列相关操作。

关于如何使用解决方案语言节点序列集合，参考[Solution_Language案例](Solution_Language_Case.md)
