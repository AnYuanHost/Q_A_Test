# FstreamSolutionLanguageTest

***

映射FstreamSolutionLanguageTest接受文件集合（FstreamSet）为定义域，布尔类型集合（BoolSet）为值域，定义为：

    Map FstreamSolutionLanguageTest(FstreamSet, &BoolSet, FstreamSolutionLanguageTestFunction);

解决方案文件语言检测映射（FstreamSolutionLanguageTest）检测定义域文件元素中解决方案语言是否符合标准，将判断结果赋值给值域布尔元素。

关于如何完成解决方案文件语言检测，参考[Solution_Language案例](Solution_Language_Case.md)
