# SortSolutionInformationDictionaries

***

映射SortSolutionInformationDictionaries接受空元素集合（NullElementSet）为定义域，解决方案信息字典集合（SolutionInformationDictionaries）为值域，定义为：

    Map SortSolutionInformationDictionaries(&NullElementSet, &SolutionInformationDictionaries, SortSolutionInformationDictionariesFunction);

解决方案信息字典排序映射（SortSolutionInformationDictionaries）在值域解决方案信息字典元素上进行排序。

关于如何完成解决方案信息字典排序，参考[Solution_Manager案例](Solution_Manager_Case.md)
