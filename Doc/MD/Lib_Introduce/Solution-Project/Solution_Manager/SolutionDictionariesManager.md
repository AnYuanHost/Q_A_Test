# SolutionDictionariesManager

***

集合SolutionDictionariesManager为解决方案字典管理的集合，定义为：

    Set SolutionDictionariesManager(EmptyElement, SolutionDictionariesManagerTest);

解决方案字典管理集合（SolutionDictionariesManager）由所有解决方案字典管理元素组成，用于解决方案字典管理相关操作。

关于如何使用解决方案字典管理，参考[Solution_Manager案例](Solution_Manager_Case.md)
