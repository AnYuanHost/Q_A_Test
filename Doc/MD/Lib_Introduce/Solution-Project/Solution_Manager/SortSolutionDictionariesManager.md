# SortSolutionDictionariesManager

***

映射SortSolutionDictionariesManager接受空元素集合（NullElementSet）为定义域，解决方案字典管理集合（SolutionDictionariesManager）为值域，定义为：

    Map SortSolutionDictionariesManager(&NullElementSet, &SolutionDictionariesManager, SortSolutionDictionariesManagerFunction);

解决方案字典管理排序映射（SortSolutionDictionariesManager）在值域解决方案字典管理元素上进行排序。

关于如何完成解决方案字典管理排序，参考[Solution_Manager案例](Solution_Manager_Case.md)
