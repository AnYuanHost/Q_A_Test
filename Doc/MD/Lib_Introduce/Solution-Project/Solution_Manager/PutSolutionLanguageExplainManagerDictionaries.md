# PutSolutionLanguageExplainManagerDictionaries

***

映射PutSolutionLanguageExplainManagerDictionaries接受元素序偶集合（OrderedPairSet）为定义域，解决方案语言描述管理字典集合（SolutionLanguageExplainManagerDictionaries）为值域，定义为：

    Map PutSolutionLanguageExplainManagerDictionaries(&OrderedPairSet, &SolutionLanguageExplainManagerDictionaries, PutSolutionLanguageExplainManagerDictionariesFunction);

解决方案语言描述管理字典存入映射（PutSolutionLanguageExplainManagerDictionaries）分析定义域元素序偶元素，其结构为：<UIntSet, SolutionLanguageExplainManager>，其信息为：<地址信息，解决方案语言描述管理元素>。按照地址信息将解决方案语言描述管理元素转交传递到值域解决方案语言描述管理字典元素指定位置。

关于如何完成解决方案语言描述管理字典存入，参考[Solution_Manager案例](Solution_Manager_Case.md)
