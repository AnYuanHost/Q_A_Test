# LogoutSolutionDictionariesManager

***

映射LogoutSolutionDictionariesManager接受超索引集合（SuperIndexSet）为定义域，解决方案字典管理集合（SolutionDictionariesManager）为值域，定义为：

    Map LogoutSolutionDictionariesManager(&SuperIndexSet, &SolutionDictionariesManager, LogoutSolutionDictionariesManagerFunction);

解决方案字典管理注销映射（LogoutSolutionDictionariesManager）将定义域超索引元素在值域解决方案字典管理元素中进行注销。

关于如何完成解决方案字典管理注销，参考[Solution_Manager案例](Solution_Manager_Case.md)
