# NewSolutionDictionariesManager

***

映射NewSolutionDictionariesManager接受空元素集合（NullElementSet）为定义域和值域，定义为：

    Map NewSolutionDictionariesManager(&NullElementSet, &NullElementSet, NewSolutionDictionariesManagerFunction);

新建解决方案字典管理映射（NewSolutionDictionariesManager）在值域空元素上默认构造解决方案字典管理元素。

关于如何完成解决方案字典管理构造，参考[Solution_Manager案例](Solution_Manager_Case.md)
