# SolutionInformationManager

***

集合SolutionInformationManager为解决方案信息管理的集合，定义为：

    Set SolutionInformationManager(EmptyElement, SolutionInformationManagerTest);

解决方案信息管理集合（SolutionInformationManager）由所有解决方案信息管理元素组成，用于解决方案信息管理相关操作。

关于如何使用解决方案信息管理，参考[Solution_Manager案例](Solution_Manager_Case.md)
