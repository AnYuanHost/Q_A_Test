# NewSolutionInformationDictionaries

***

映射NewSolutionInformationDictionaries接受空元素集合（NullElementSet）为定义域和值域，定义为：

    Map NewSolutionInformationDictionaries(&NullElementSet, &NullElementSet, NewSolutionInformationDictionariesFunction);

新建解决方案信息字典映射（NewSolutionInformationDictionaries）在值域空元素上默认构建解决方案信息字典元素。

关于如何完成解决方案信息字典构造，参考[Solution_Manager案例](Solution_Manager_Case.md)
