# SolutionInformationDictionariesToData

***

映射SolutionInformationDictionariesToData接受解决方案信息字典集合（SolutionInformationDictionaries）为定义域，空元素集合（NullElementSet）为值域，定义为：

    Map SolutionInformationDictionariesToData(&SolutionInformationDictionaries, &NullElementSet, SolutionInformationDictionariesToDatFunction);

解决方案信息字典数据转化映射（SolutionInformationDictionariesToData）分析定义域解决方案信息字典信息，将其转化为数据流信息，置于值域空元素上。

关于如何完成解决方案信息字典数据转化，参考[Solution_Manager案例](Solution_Manager_Case.md)
