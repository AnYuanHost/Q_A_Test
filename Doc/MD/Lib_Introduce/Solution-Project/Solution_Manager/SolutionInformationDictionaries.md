# SolutionInformationDictionaries

***

集合SolutionInformationDictionaries为解决方案信息字典的集合，定义为：

    Set SolutionInformationDictionaries(EmptyElement, SolutionInformationDictionariesTest);

解决方案信息字典集合（SolutionInformationDictionaries）由解决方案信息字典元素组成，用于解决方案信息字典相关操作。

关于如何使用解决方案信息字典，参考[Solution_Manager案例](Solution_Manager_Case.md)
