# NewSolutionInformationManager

***

映射NewSolutionInformationManager接受空元素集合（NullElementSet）为定义域和值域，定义为：

    Map NewSolutionInformationManager(&NullElementSet, &NullElementSet, NewSolutionInformationManagerFunction);

新建解决方案信息管理映射（NewSolutionInformationManager）在值域空元素上默认构造坚决方案信息管理元素。

关于如何完成解决方案信息管理构建，参考[Solution_Manager案例](Solution_Manager_Case.md)
