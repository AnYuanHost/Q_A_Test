# ReleaseKRSPollingDrivenZero

***

映射ReleaseKRSPollingDrivenZero接受空元素集合（NullElementSet）为定义域，知识库零轮询机集合（KRSPollingDrivenZero）为值域，定义为：

    Map ReleaseKRSPollingDrivenZero(&NullElementSet, &KRSPollingDrivenZero, ReleaseKRSPollingDrivenZeroFunction);

释放知识库零轮询机映射（ReleaseKRSPollingDrivenZero）分析并释放值域知识库零轮询机元素。

关于如何完成知识库零轮询机释放，参考[KRSPollingDrivenZero案例](KRSPollingDrivenZero_Case.md)
