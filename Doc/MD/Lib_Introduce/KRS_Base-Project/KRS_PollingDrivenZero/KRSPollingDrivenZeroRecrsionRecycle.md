# KRSPollingDrivenZeroRecursionRecycle

***

映射KRSPollingDrivenZeroRecursionRecycle接受空元素集合（NullElementSet）为定义域，知识库零轮询机集合（KRSPollingDrivenZero）为值域，定义为：

    Map KRSPollingDrivenZeroRecursionRecycle(&NullElementSet, &KRSPollingDrivenZero, KRSPollingDrivenZeroRecursionRecycleFunction);

知识库零轮询机递归回收映射（KRSPollingDrivenZeroRecursionRecycle）在值域知识库零轮询机元素上完成内循环问答流图的回收映射。

关于如何完成知识库零轮询机递归回收，参考[KRSPollingDrivenZero案例](KRSPollingDrivenZero_Case.md)
