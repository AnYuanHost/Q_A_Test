# KRSPollingDrivenZeroReplace

***

映射KRSPollingDrivenZeroReplace接受空元素集合（NullElementSet）为定义域，知识库零轮询机集合（KRSPollingDrivenZero）为值域，定义为：

    Map KRSPollingDrivenZeroReplace(&NullElementSet, &KRSPollingDrivenZero, KRSPollingDrivenZeroReplaceFunction);

知识库零轮询机替换映射（KRSPollingDrivenZeroReplace）在值域知识库零轮询机元素的外循环问答流图中，通过解决方案管理提供的信息来完成子图替换。

关于如何完成知识库零轮询机替换，参考[KRSPollingDrivenZero案例](KRSPollingDrivenZero_Case.md)
