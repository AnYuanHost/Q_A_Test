# KRSPollingDrivenZeroConstructMap

***

映射KRSPollingDrivenZeroConstructMap接受元素序偶集合（OrderedPairSet）为定义域，空元素集合（NullElementSet）为值域，定义为：

    Map KRSPollingDrivenZeroConstructMap(&OrderedPairSet, &NullElementSet, KRSPollingDrivenZeroConstructFunction);

知识库零轮询机构造映射映射（KRSPollingDrivenZeroDConstructMap）分析定义域元素序偶元素，其结构为：<SuperIndexSet, KRSPollingDrivenZero>，其信息为：<映射检索信息，知识库零轮询机元素>。在知识库零轮询机元素提供的检索环境中完成对映射信息的检索，将检索结果置于值域空元素上。

关于如何完成知识库零轮询机映射检索，参考[KRSPollingDrivenZero案例](KRSPollingDrivenZero_Case.md)
