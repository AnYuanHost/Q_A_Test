# KRSPollingDrivenZeroRecycle

***

映射KRSPollingDrivenZeroRecycle接受空元素集合（NullElementSet）为定义域，知识库零轮询机集合（KRSPollingDrivenZero）为值域，定义为：

    Map KRSPollingDrivenZeroRecycle(&NullElementSet, &KRSPollingDrivenZero, KRSPollingDrivenZeroRecycleFunction);

知识库零轮询机回收映射（KRSPollingDrivenZeroRecycle）在值域知识库零轮询机的外循环问答流图中完成节点回收。

关于如何完成知识库零轮询机回收，参考[KRSPollingDrivenZero案例](KRSPollingDrivenZero_Case.md)
