# SetKRSPollingDrivenZeroRecursionMap

***

映射SetKRSPollingDrivenZeroRecursionMap接受问答流图集合（QuestionFlowMap）为定义域，知识库零轮询机集合（KRSPollingDrivenZero）为值域，定义为：

    Map SetKRSPollingDrivenZeroRecursionMap(&QuestionFlowMap, &KRSPollingDrivenZero, SetKRSPollingDrivenZeroRecursionMapFunction);

知识库零轮询机迭代图设置映射（SetKRSPollingDrivenZeroRecursionMap）将定义域问答流图元素设置到值域知识库零轮询机元素的指定位置上。

关于如何完成知识库零轮询机迭代问答流图设置，参考[KRSPollingDrivenZero案例](KRSPollingDrivenZero_Case.md)
