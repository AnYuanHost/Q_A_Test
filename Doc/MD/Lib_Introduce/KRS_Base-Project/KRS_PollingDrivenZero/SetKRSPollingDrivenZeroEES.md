# SetKRSPollingDrivenZeroEES

***

映射SetKRSPollingDrivenZeroEES接受元素环境检索器集合（ElementEnvironmentSearcher）为定义域，知识库零轮询机集合（KRSPollingDrivenZero）为值域，定义为：

    Map SetKRSPollingDrivenZeroEES(&ElementEnvironmentSearcher, &KRSPollingDrivenZero, SetKRSPollingDrivenZeroEESFunction);

知识库零轮询机设置元素环境检索器映射（SetKRSPollingDrivenZeroEES）将定义域元素环境检索器元素设置到值域知识库零轮询机元素的指定位置。

关于如何完成知识库零轮询机元素环境检索器设置，参考[KRSPollingDrivenZero案例](KRSPollingDrivenZero_Case.md)
