# KRSPollingDrivenZeroPushBack

***

映射KRSPollingDrivenZeroPushBack接受空元素集合（NullElementSet）为定义域，知识库零轮询机集合（KRSPollingDrivenZero）为值域，定义为：

    Map KRSPollingDrivenZeroPushBack(&NullElementSet, &KRSPollingDrivenZero, KRSPollingDrivenZeroPushBackFunction);

知识库零轮询机后推映射（KRSPollingDrivenZeroPushBack）在值域知识库零轮询机元素中对外循环问答流图进行后推。

关于如何完成知识库零轮询机后推，参考[KRSPollingDrivenZero案例](KRSPollingDrivenZero_Case.md)
