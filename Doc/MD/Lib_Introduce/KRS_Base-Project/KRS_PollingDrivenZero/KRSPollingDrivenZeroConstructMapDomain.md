# KRSPollingDrivenZeroConstructMapDomain

***

映射KRSPollingDrivenZeroConstructMapDomain接受元素序偶集合（OrderedPairSet）为定义域，空元素集合（NullElementSet）为值域，定义为：

    Map KRSPollingDrivenZeroConstructMapDomain(&OrderedPairSet, &NullElementSet, KRSPollingDrivenZeroConstructMapDomainFunction);

知识库零轮询机构造映射域映射（KRSPollingDrivenZeroConstructMapDomain）分析定义域序偶元素，其结构为：<ElementOrderSet, KRSPollingDrivenZero>，其信息为：<待解释域信息，知识库零轮询机元素>。知识库零轮询机提供解释构造过程所需的各种信息，将待解释域信息进行解释，构造结果置于值域空元素上。

关于如何完成知识库零轮询机映射域构造，参考[KRSPollingDrivenZero案例](KRSPollingDrivenZero_Case.md)
