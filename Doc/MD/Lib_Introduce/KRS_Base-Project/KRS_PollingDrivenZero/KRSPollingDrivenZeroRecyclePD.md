# KRSPollingDrivenZeroRecyclePD

***

映射KRSPollingDrivenZeroRecyclePD接受空元素集合（NullElementSet）为定义域，知识库零轮询机集合（KRSPollingDrivenZero）为值域，定义为：

    Map KRSPollingDrivenZeroRecyclePD(&NullElementSet, &KRSPollingDrivenZero, KRSPollingDrivenZeroRecyclePDFunction);

知识库零轮询机回收轮询机映射（KRSPollingDrivenZeroRecyclePD）在值域知识库零轮询机中回收轮询机。

关于如何完成知识库零轮询机回收轮询机，参考[KRSPollingDrivenZero案例](KRSPollingDrivenZero_Case.md)
