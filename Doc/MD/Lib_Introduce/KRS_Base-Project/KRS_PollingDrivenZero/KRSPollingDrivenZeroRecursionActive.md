# KRSPollingDrivenZeroRecursionActive

***

映射KRSPollingDrivenZeroRecursionActive接受空元素集合（NullElementSet）为定义域，知识库零轮询机集合（KRSPollingDrivenZero）为值域，定义为：

    Map KRSPollingDrivenZeroRecursionActive(&NullElementSet, &KRSPollingDrivenZero, KRSPollingDrivenZeroRecursionActiveFunction);

知识库零轮询机递归执行映射（KRSPollingDrivenZeroRecursionActive）在值域知识库零轮询机元素上执行内循环问答流图的执行映射。

关于如何完成知识库零轮询机递归执行，参考[KRSPollingDrivenZero案例](KRSPollingDrivenZero_Case.md)
