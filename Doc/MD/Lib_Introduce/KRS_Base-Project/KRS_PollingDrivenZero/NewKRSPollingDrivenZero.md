# NewKRSPollingDrivenZero

***

映射NewKRSPollingDrivenZero接受空元素集合（NullElementSet）为定义域和值域，定义为：

    Map NewKRSPollingDrivenZero(&NullElementSet, &NullElementSet, NewKRSPollingDrivenZeroFunction);

新建知识库零轮询机映射（NewKRSPollingDrivenZero）在值域空元素上默认构造知识库零轮询机元素。

关于如何完成知识库零轮询机构造，参考[KRSPollingDrivenZero案例](KRSPollingDrivenZero_Case.md)
