# KRSPollingDrivenZeroDestructMapDomain

***

映射KRSPollingDrivenZeroDestructMapDomain接受元素序偶集合（OrderedPairSet）为定义域，元素序列集合（ElementOrderSet）为值域，定义为：

    Map KRSPollingDrivenZeroDestructMapDomain(&OrderedPairSet, &ElementOrderSet, KRSPollingDrivenZeroDestructMapDomainFunction);

知识库零轮询机析构域映射域映射（KRSPollingDrivenZeroDestructMapDomain）分析定义域元素序偶元素，其结构为：<ElementOrderSet, KRSPollingDrivenZero>，其信息为：<待解释域信息，知识库零轮询机元素>。知识库零轮询机元素提供析构所需的各种信息，将待解释域信息进行解释，以此析构值域元素序列元素数。

关于如何完成知识库零轮询机映射域析构，参考[KRSPollingDrivenZero案例](KRSPollingDrivenZero_Case.md)
