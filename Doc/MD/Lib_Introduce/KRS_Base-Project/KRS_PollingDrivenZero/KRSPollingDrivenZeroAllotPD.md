# KRSPollingDrivenZeroAllotPD

***

映射KRSPollingDrivenZeroAllotPD接受空元素集合（NullElementSet）为定义域，知识库零轮询机集合（KRSPollingDrivenZero）为值域，定义为：

    Map KRSPollingDrivenZeroAllotPD(&NullElementSet, &KRSPollingDrivenZero, KRSPollingDrivenZeroAllotPDFunction);

知识库零轮询机申请轮询机映射（KRSPollingDrivenZeroAllotPD）在值域知识库零轮询机中申请新的轮询机。

关于如何完成知识库零轮询机申请轮询机，参考[KRSPollingDrivenZero案例](KRSPollingDrivenZero_Case.md)
