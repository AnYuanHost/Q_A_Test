# InTestKRSPollingDrivenZero_MapSearcher

***

映射InTestKRSPollingDrivenZero_MapSearcher接受超索引集合（SuperIndexSet）为定义域，布尔类型集合（BoolSet）为值域，定义为：

    Map InTestKRSPollingDrivenZero_Map(&SuperIndexSet, &BoolSet, InTestKRSPollingDrivenZero_MapSearcherFunction);

知识库零轮询机映射检索器存在测试映射（InTestKRSPollingDrivenZero_MapSearcher）检测定义域超索引元素是否在指定元素（KRSPollingDrivenZero_MapSearcher）中，并将检测结果置于值域布尔元素上。

关于如何完成知识库零轮询机映射检索器存在测试，参考[KRS_PollingDrivenZero案例](KRS_PollingDrivenZero_Case.md)。
