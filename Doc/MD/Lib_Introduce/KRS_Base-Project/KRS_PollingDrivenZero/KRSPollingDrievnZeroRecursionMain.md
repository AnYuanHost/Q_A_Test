# KRSPollingDrivenZeroRecursionMain

***

映射KRSPollingDrivenZeroRecursionMain接受空元素集合（NullElementSet）为定义域，知识库零轮询机集合（KRSPollingDrivenZero）为值域，定义为：

    Map KRSPollingDrivenZeroRecursionMain(&NullElementSet, &KRSPollingDrivenZero, KRSPollingDrivenZeroRecursionMainFunction);

知识库零轮询机递归主映射（KRSPollingDrivenZeroRecursionMain）在值域知识库零轮询机元素上执行内循环主映射。

关于如何执行知识库零轮询机主映射，参考[KRSPollingDrivenZero案例](KRSPollingDrivenZero_Case.md)
