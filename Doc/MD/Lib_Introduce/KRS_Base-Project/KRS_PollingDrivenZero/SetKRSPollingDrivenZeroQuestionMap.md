# SetKRSPollingDrivenZeroQuestionMap

***

映射SetKRSPollingDrivenZeroQuestionMap接受问答流图集合（QuestionFlowMap）为定义域，知识库零轮询机集合（KRSPollingDrivenZero）为值域，定义为：

    Map SetKRSPollingDrivenZeroQuestionMap(&QuestionFlowMap, &KRSPollingDrivenZero, SetKRSPollingDrivenZeroQuestionMapFunction);

知识库零轮询机问答流图设置映射（SetKRSPollingDrivenZeroQuestionMap）将定义域问答流图元素设置到值域知识库零轮询机问答流图的指定位置。

关于如何完成知识库零轮询机问答流图设置，参考[KRSPollingDrivenZero案例](KRSPollingDrivenZero_Case.md)
